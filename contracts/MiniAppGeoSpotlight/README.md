# Geo Spotlight

Pay to highlight locations on a global map with attention-based economy.

## 中文说明

地球放大镜 - 在全球地图上支付以突出显示位置，基于注意力的经济模型

### 功能特点

- 在地图上创建地理位置聚光灯
- 出价越高，聚光灯越大越显眼
- 所有用户可见的全球地图
- 1小时持续时间，可通过追加投资延长

### 使用方法

1. 支付至少0.1 GAS创建地理聚光灯（包含位置和消息）
2. 其他用户可追加投资增强聚光灯亮度
3. 每次追加投资延长1小时有效期
4. 查看全球地图上的所有活跃聚光灯

### 技术细节

- **最低费用**: 0.1 GAS
- **持续时间**: 1小时（可延长）
- **消息长度**: 最多100字符
- **存储**: 位置坐标、消息、总投资、过期时间

## English

### Features

- Create geographic spotlights on global map
- Higher bids get bigger, more visible spotlights
- Global map visible to all users
- 1-hour duration, extendable with boosts

### Usage

1. Pay minimum 0.1 GAS to create geo spotlight (with location and message)
2. Others can boost spotlight to increase brightness
3. Each boost extends duration by 1 hour
4. View all active spotlights on global map

### Technical Details

- **Minimum Fee**: 0.1 GAS
- **Duration**: 1 hour (extendable)
- **Message Length**: Max 100 characters
- **Storage**: Location coordinates, message, total boost, expiry time

## Technical

- **Contract**: MiniAppGeoSpotlight
- **Category**: Social/Location
- **Permissions**: Full contract permissions
- **Assets**: GAS
- **Services**: None (pure on-chain logic)
