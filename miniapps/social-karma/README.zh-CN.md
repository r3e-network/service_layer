# Social Karma 社交因果

Neo N3 的链上声誉和因果系统。

## 概述

Social Karma 追踪并奖励积极的社区贡献。通过有益的行动建立链上声誉并获得因果点数。

## 功能

- **因果点数**：积极行为获得点数
- **声誉分数**：建立可验证的链上声誉
- **背书**：给予和接受同伴背书
- **徽章**：解锁成就徽章
- **排行榜**：社区排名

## 工作原理

1. 连接您的钱包
2. 参与社区活动
3. 接受同伴的背书
4. 随着时间建立您的因果分数

## 技术细节

- **类别**: 社交
- **网络**: Neo N3 主网

## 开发

```bash
cd apps/social-karma
pnpm install
pnpm dev
```
