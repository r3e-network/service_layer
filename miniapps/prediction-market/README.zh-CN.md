# Prediction Market 预测市场

Neo N3 上的去中心化预测市场。

## 概述

创建并参与真实世界事件的预测市场。下注预测结果，准确预测可获得奖励。

## 功能

- **创建市场**：为任何事件启动预测市场
- **二元与多结果**：是/否或多选题市场
- **自动结算**：基于预言机的结果验证
- **流动性池**：AMM 风格的市场做市
- **排行榜**：追踪顶级预测者

## 工作原理

1. 浏览活跃的预测市场
2. 购买您预测结果的份额
3. 等待事件结算
4. 如果预测正确，领取奖金

## 技术细节

- **类别**: 金融
- **网络**: Neo N3 主网

## 开发

```bash
cd apps/prediction-market
pnpm install
pnpm dev
```
