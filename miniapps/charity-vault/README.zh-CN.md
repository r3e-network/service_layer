# Charity Vault 慈善金库

Neo N3 区块链上的透明慈善捐赠。

## 概述

Charity Vault 实现透明、可问责的慈善捐赠，具有完整的链上追踪功能。捐赠者可以为已验证的事业做出贡献，并准确追踪资金的分配情况。

## 功能

- **透明捐赠**：所有贡献记录在链上
- **资金追踪**：实时查看资金分配
- **已验证事业**：精选的合法慈善组织列表
- **税收收据**：自动生成捐赠记录
- **多币种**：支持 GAS 和 NEP-17 代币

## 工作原理

1. 浏览已验证的慈善事业
2. 选择捐赠金额和币种
3. 通过钱包确认交易
4. 追踪您的贡献及其影响

## 技术细节

- **类别**: 金融
- **网络**: Neo N3 主网
- **权限**: invoke:primary, read:blockchain

## 开发

```bash
cd apps/charity-vault
pnpm install
pnpm dev
```
