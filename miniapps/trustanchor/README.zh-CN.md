# TrustAnchor MiniApp

Neo N3 治理的非营利性投票委托。

## 描述

TrustAnchor 允许 NEO 持有者将其投票权委托给具有良好声誉和积极贡献的候选人。平台采用零费用模式，确保 100% 的 GAS 奖励归质押者所有。

## 核心理念

> "放大积极贡献者的声音。为声誉投票，而非利润。"

TrustAnchor 旨在促进优质治理。GAS 奖励是自然的激励，但我们真正的目的是确保 Neo N3 由积极、有信誉的贡献者治理。

## 功能

- **质押 NEO** - 通过质押 NEO 参与治理
- **为声誉投票** - 将投票权委托给有良好记录的候选人
- **零费用** - 100% 的 GAS 奖励归质押者所有
- **透明** - 所有操作都在链上

## 使用

### 开发

```bash
cd apps/trustanchor
pnpm dev
```

### 构建

```bash
pnpm build
```

## 架构

### 可组合函数

- `useTrustAnchor` - 质押、委托和奖励的核心业务逻辑

### 组件

- `StatsGrid` - 以网格布局显示质押统计
- `StakeForm` - 质押/解押输入表单（即将推出）

### 智能合约

TrustAnchor 智能合约处理：
- NEO 的质押和解押
- 向候选人委托投票
- GAS 奖励分配

合约地址：
- **测试网**: `0x0000000000000000000000000000000000000000`
- **主网**: `0x0000000000000000000000000000000000000000`

## 权限

| 权限 | 状态 | 用途 |
|------|------|------|
| `governance` | ✅ | 投票委托和候选人选择 |
| `payments` | ❌ | 不需要（无支付处理） |
| `rng` | ❌ | 不需要 |
